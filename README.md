# Open NAMAE client

Open NAMAE clientはGNU/LinuxとWindowsの双方で動作可能な、「お名前.com DDNSクライアント」の非公式オープンソース実装です。  
アプリ本体はcronから実行可能なようCUIとしつつも、設定画面はGUIとすることで公式実装からの切り替えを容易にしています。
